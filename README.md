# Klondike Solitaire

A modern implementation of Klondike Solitaire built with HTML, CSS, and JavaScript.

## Features

- Click-to-move gameplay
- Draw 3 stock
- Undo
- New Game
- Automatic foundation moves
- Stack movement
- Responsive design (in progress)

## Planned Features

- Card animations
- Statistics
- Timer
- Themes
- Sound effects
- Mobile support
